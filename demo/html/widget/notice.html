站点通知

提供弹出和关闭站点通知的 API。

每个页面只可弹出一个站点通知。即，在弹出之前应检测是否已有其它通知正在显示；如果页面已有一个通知，则弹出操作无效。

API

弹出： DPL.siteNotice.show(oCfg);

oCfg = {
	html: '<.../>',
	actionClose: '#xxx',
	hideCloseBtn: b,
	id: '',
	type: 'info'  //'error', 'notice', 'warning', 'tip'...
};

关闭： DPL.siteNotice.hide();

检测是否显示： DPL.siteNotice.isShown
